# Privacy Policy

This privacy policy applies to the open source HamsterList application.

## Information Collection and Use

HamsterList does not collect any personal data.

If you choose to use a server for syncing shopping lists, the server provider might collect the following data:

- username
- shopping list data (items, categories, orders, change history)

HamsterList does not provide any servers, nor does it recommend any in the app.
HamsterList is not liable for servers introduced by the user for syncing.
Please consult with the server provider on which data they store, how long they store your data and how to delete your data.

You can also host your own server using the open source [HamsterList API](https://github.com/tstehr/HamsterList).

## Changes

This privacy policy was last updated on November 24th 2025.

If you have any questions, please contact me on [Github](https://github.com/Evylon/HamsterList-Native/issues).

# Datenschutzerklärung

Diese Datenschutzerklärung gilt für die Open Source App HamsterList

## Datenspeicherung und -nutzung

HamsterList sammelt keine persönlichen Daten.

Falls Sie sich entscheiden einen Server für das Synchronisieren von Einkaufslisten zu benutzen, ist es möglich, dass dieser Server unter anderem die folgenden Daten speichert:

- Nutzername
- Details der Einkaufsliste (Einträge, Kategorien, Sortierungen, Änderungen)

HamsterList stellt keine Server bereit, es werden keine Server in der App vorgeschlagen.
HamsterList ist nicht verantwortlich für Server, die der Nutzer für Synchronisierung angibt.
Bitte konsultieren Sie den jeweiligen Anbieter eines Servers, inwiefern Daten gespeichert werden, für wie lange und wie diese gelöscht werden können.

Sie können auch Ihren eigenen Server mit der [HamsterList API](https://github.com/tstehr/HamsterList) hosten.
